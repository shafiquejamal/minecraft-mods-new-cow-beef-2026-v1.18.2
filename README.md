# New Cow Beef Forge 1.18.2

Starter skeleton for a Minecraft Forge `1.18.2` mod written in Kotlin.

## Build

```bash
./gradlew build
```

## Run Dev Client

```bash
./gradlew runClient
```

## Project Notes

- Mod ID: `new_cow_beef`
- Main class: `com.github.shafiquejamal.newcowbeef.NewCowBeefMod`
- Kotlin for Forge is included and ready for additional registries, items, blocks, and events.
